# packages

This directory contains shared libraries used across WEMA applications.

| Package | Description |
|---|---|
| `packages/contracts` | Shared Zod schemas and TypeScript types |
| `packages/scoring` | Clinical scoring functions (EPDS, PHQ-2, PHQ-9) |
| `packages/clinical-rules` | Routing and risk classification logic |
| `packages/database` | Prisma client and database access layer |
| `packages/ui` | Shared React UI components |
| `packages/testing` | Shared test utilities and fixtures |
